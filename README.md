# HomeService Backend (Node + Express + MySQL + Sequelize)

## Features
- Auth (Register/Login) with JWT
- Role based access (Admin/User)
- Services CRUD (Admin)
- Booking flow (Create, My bookings, Admin confirm, Cancel)
- Email notifications (Nodemailer)
- Realtime notifications (Socket.io)
- Testing (Jest + Supertest)

## Setup
1) Install
npm install

2) Env
Create `.env` (and `.env.test` for tests)

3) Run
npm start

4) Tests
npm test
